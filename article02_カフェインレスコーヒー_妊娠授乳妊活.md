# 妊娠中・授乳中でもコーヒーを楽しみたい。カフェインレスの正しい付き合い方

「妊娠がわかった瞬間から、コーヒーをやめました」

そういう方、本当に多いんです。大切な赤ちゃんのために、大好きなコーヒーを我慢する。その気持ちはとても自然ですし、すごく真剣に向き合っている証拠だと思います。

でも、「完全にゼロにしないといけない」というのは、実は少し厳しすぎる話かもしれません。カフェインとの付き合い方には、もう少し柔軟な選択肢があります。この記事では、妊娠中・授乳中の方が「どれくらいなら大丈夫なのか」「カフェインレスコーヒーはどう使えばいいのか」を、できるだけわかりやすくお伝えします。

不安な気持ちに寄り添いながら、一緒に考えていきましょう。

## 「カフェインは絶対ダメ」じゃなかった。専門機関の見解

まず、一番気になるところからお伝えします。

**WHO（世界保健機関）やEFSA（欧州食品安全機関）は、妊娠中のカフェイン摂取を「1日200mg以内」であれば大きなリスクはないとしています。**

「え、ゼロじゃないの？」と思った方もいるかもしれません。完全にゼロにする必要はなく、量をコントロールすることが大事、というのが専門機関の共通した見解です。

では、200mgというのはどれくらいかというと——ドリップコーヒー（150ml）で約2杯分です。毎日2杯飲み続けるのは少し多いですが、1杯程度なら範囲内ということになります。

### 1日のカフェイン、実はこんなに積み重なっています

ここで気をつけてほしいのが、カフェインはコーヒーだけに入っているわけではないということ。知らないうちに積み重なっていることがあります。

| 飲み物・食べ物 | 目安量 | カフェイン量 |
|---|---|---|
| ドリップコーヒー | 150ml | 約90mg |
| カフェインレスコーヒー | 150ml | 約2〜9mg |
| 紅茶 | 150ml | 約45mg |
| 煎茶（緑茶） | 150ml | 約30mg |
| ほうじ茶 | 150ml | 約15mg |
| コーラ | 350ml | 約34mg |
| チョコレート | 50g | 約10〜30mg |

たとえば「朝に紅茶1杯（45mg）、昼に緑茶1杯（30mg）、おやつにチョコ少し（15mg）」だけで、合計90mg近くになります。そこにコーヒーを1杯追加すると、すでに180mg。気づかないうちに200mgに近づいていることも十分あります。

「何を・何杯飲んでいるか」を一度振り返ってみると、自分の1日のカフェイン量が見えてきますよ。

## 妊娠中・授乳中にカフェインレスコーヒーを選ぶメリット

カフェインレスコーヒーは、この時期の強い味方になります。理由はシンプルで、**1杯あたりのカフェインが2〜9mgと極めて少ないから**です。

通常のコーヒーなら1〜2杯で200mgに近づいてしまいますが、カフェインレスなら20杯以上飲まないと上限に達しません。現実的に飲みすぎる心配がほぼないレベルです。

それだけじゃなく、もう一つ大事なことがあります。**コーヒーの時間を手放さなくていい、ということです。**

妊娠中や授乳中は、体の変化や生活の制限が多くて、気持ちが張り詰めることも多いですよね。好きなコーヒーの香りに包まれる時間、ほっと一息つく時間——それが「我慢しなきゃ」になってしまうのは、心への負担も大きいです。

カフェインレスコーヒーがあれば、その時間はそのまま守れます。体にも心にも、無理をさせなくていいんです。

## 「お医者さんにカフェインを控えてと言われた」方へ

このケースで一番大切なのは、**「完全にゼロ」と指示されているのか、「量を減らして」という意味なのかを確認することです。**

「控えてください」という言葉は、医師によってニュアンスが違うことがあります。血圧や睡眠の問題で言われている場合、「量を減らせばOK」のケースも多いです。カフェインレスコーヒーへの置き換えで対応できることもあります。

一方、体の状態によっては「カフェインレスの微量でも」という場合もゼロではありません。心配な方は、次の受診のときに「カフェインレスコーヒーなら飲んでもいいですか？」と一言聞いてみてください。それだけで、気持ちがずいぶん楽になりますよ。

カフェインについての判断は、体の状態や状況によって本当に人それぞれです。「自分はどうすればいいんだろう」と迷ったときは、ぜひ担当の医師や助産師に相談してみてください。「カフェインレスコーヒーに置き換えるのはどうですか？」と具体的に聞くと、より明確なアドバイスをもらいやすいですよ。

## よくある質問

**Q. 妊娠中、カフェインレスコーヒーは1日何杯まで飲めますか？**

カフェインレスコーヒー1杯あたりのカフェインは約2〜9mgです。1日200mgという目安から考えると、20杯以上飲まない限り超える心配はほぼありません。ただし、他の飲み物との合計で考えることを忘れずに。

**Q. 授乳中に飲んでも、母乳に影響しますか？**

カフェインは母乳に微量移行します。ただしカフェインレスコーヒーの場合、1杯のカフェイン量は極めて少ないため、通常の範囲で飲む分には大きな影響はないとされています。心配な場合は授乳直後に飲むと、次の授乳までに体内で代謝される時間が確保できます。

**Q. カフェインレスコーヒーと麦茶、どちらがいいですか？**

目的によって変わります。カフェインを完全にゼロにしたいなら麦茶などのノンカフェイン飲料が確実です。コーヒーの香りや風味を楽しみながらカフェインを減らしたいなら、カフェインレスコーヒーが向いています。どちらが「正解」ということはなく、自分が続けやすい方を選ぶのが一番です。

**Q. カフェインレス紅茶とカフェインレスコーヒー、どちらを選べばいいですか？**

紅茶派の方はカフェインレス紅茶、コーヒー派の方はカフェインレスコーヒーで問題ありません。カフェイン量の観点ではほぼ同じです。好きな方を選んで、我慢のない毎日を続けましょう。

## まとめ：我慢しすぎなくて、大丈夫です

- 専門機関は「1日200mg以内なら大きなリスクはない」としています。完全ゼロが必須ではありません
- カフェインはコーヒー以外にも紅茶・緑茶・チョコなどに含まれるので、1日のトータルで考えましょう
- カフェインレスコーヒーは1杯2〜9mgなので、飲みすぎる心配がほぼありません
- 「控えて」と言われた方は、カフェインレスへの置き換えが可能か医師や助産師に確認してみましょう
- 迷ったときはひとりで抱え込まず、プロに相談するのが一番の近道です

コーヒーが好きなあなたが、この時期も好きな時間を守れますように。カフェインレスコーヒーは、そのための選択肢のひとつです。
